*** Installing Libraries***

Download apache-jena-fuseki-4.5.0
With IntelliJ, install the library from Project Structure -> add library
Download apache-maven-3.8.5-bin
Unzip and add the folder to usr/local folder
open terminal and run
    export PATH=/usr/local/apache-maven-3.8.5/bin:$PATH
    mvn -version

mvn -version will give you some info if it worked

*** Maven + Dependencies ***
Maven is used to manage our dependencies. These will all be stored in a POM.xml file

run this in the project folder -> this will setup maven
    mvn archetype:generate -DgroupId=com.mycompany.app -DartifactId=my-app -DarchetypeArtifactId=maven-archetype-quickstart -DarchetypeVersion=1.4 -DinteractiveMode=false
In the POM.xml file, change the groupId and artifactId to match your project. Move the POM.xml file into your main directory.
run this:
    mvn package
